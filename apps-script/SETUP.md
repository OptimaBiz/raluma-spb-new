# Backend формы Raluma

1. Откройте https://script.google.com/create под аккаунтом `anton.karapetian@gmail.com`.
2. Замените содержимое `Code.gs` кодом из файла `Code.gs` этой папки.
3. В настройках проекта включите показ файла манифеста и замените `appsscript.json` файлом из этой папки.
4. В редакторе выберите функцию `setup`, нажмите «Выполнить» и подтвердите разрешения Google.
5. Нажмите «Начать развертывание» → «Новое развертывание» → «Веб-приложение».
6. Выберите «Выполнять как: Я» и «У кого есть доступ: Все».
7. Скопируйте адрес, который заканчивается на `/exec`, и передайте его разработчику сайта.

После подключения сайт начнёт подготавливать и загружать файл сразу после выбора. При отправке заявки временный файл будет приложен к письму в Gmail и удалён из Google Drive.
